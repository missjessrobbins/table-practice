CSS Selectors

In this article we will go over some common CSS Selectors and their functions

Selecting HTML Elements

HTML
(Example)		 CSS Selector
(Example)		 Purpose
By HTML Tag <h1> h1 {

}		 By selecting elements based on their HTML tag, the value applies to all elements with that tag.
By Class <h1 class="uppercase"> .uppercase {

}		 By selecting elements by their class name, you can apply a CSS rule to a more specific range of HTML elements
By ID <input type="text" id="email"> #email {

}		 By selecting elements by their ID, you can be even more specific about which elements you wish to style

Chaining Selectors

If you wanted to select, for example, only the <p> elements nested under the <footer> element, you would use the following CSS selector:

footer p {

color: blue;

}
